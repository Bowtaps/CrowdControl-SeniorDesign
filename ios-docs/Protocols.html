  • UserProfileModel

    Model protocol representing the public profile of any user in the app. This model is intended to be an adequate interface for interacting with other users of the app, as it contains sufficient information about the other user to communicate with them.

  • UserModel

    Model protocol representing a user who is using the app. This model is not intended to represent any user other than the one currently using the device, as it contains information that should not be shared with users other than the owner of the data.

  • BaseModel

    Base model protocol, which is to be implemented by all model classes in the project. Provides access to basic information about the model, such as:

    • Unique identifier
    • Creation timestamp
    • Last updated timestamp
    • Flag indicating modification since last save
    • Methods for loading from and saving to storage
