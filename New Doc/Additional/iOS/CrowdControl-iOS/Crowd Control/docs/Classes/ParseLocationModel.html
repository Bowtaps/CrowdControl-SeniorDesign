  • fetchLocationsForUser(_:)

    Fetches all locations sent from storage that were sent to the given user.

    This is a blocking function that can take a while to complete. This function should not be called on the UI thread.

    Parameter

    Parameter user: The ParseUserProfileModel object for which to fetch ParseLocationModel objects from storage.

    Returns

    An array of ParseLocationModel objects if the operation was successful, or nil if the operation was unsuccessful. It is possible for the returned array to be empty if no matching ParseLocationModel objects are found.
